# Data directory

This directory stores data access notes, population labels, and exact target coordinates used in the manuscript.

Files:

- access/GENOTYPE_RETRIEVAL.md: genotype retrieval and reproducibility notes.
- global25_targets.csv: exact Ashkenazi_Germany and Italian_Jew scaled Global25 coordinates used in the manuscript.
- population_labels_qpadm.csv: target, source, and right-set labels used in qpAdm analyses.
- population_labels_fst.csv: target and comparator labels used in FST analyses.
- qpadm_right_set.txt: qpAdm right-set population list.
- fst_comparator_populations.txt: FST comparator population list.
- external/: place downloaded public comparator coordinate/genotype metadata files here when reproducing locally.
