# FST web-platform reproduction notes

The FST comparison reported in the manuscript was run in the IllustrativeDNA Admix Lab web interface using Human Origins v62.0. No local FST shell script was used for the submitted FST results.

To reproduce the reported FST run, use the exact target, comparator labels, and Adjust Pseudohaploids setting recorded in:

- ../ILLUSTRATIVEDNA_REPRODUCTION_PROTOCOL.md
- ../models/fst_comparisons.tsv
- ../data/fst_comparator_populations.txt
- ../outputs/raw/illustrativedna/illustrativedna_fst_jew_ashkenazi_raw_output.txt

The platform output reported HO.geno with 21,945 samples and 584,131 SNPs, and allele frequencies calculated from 118 samples in 13 populations.
