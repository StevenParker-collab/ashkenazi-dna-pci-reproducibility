# qpAdm web-platform reproduction notes

The qpAdm models reported in the manuscript were run in the IllustrativeDNA Admix Lab web interface using Human Origins v62.0. No local qpAdm shell script was used for the submitted qpAdm results.

To reproduce the reported qpAdm runs, use the exact target, source, right-set, and allsnps settings recorded in:

- ../ILLUSTRATIVEDNA_REPRODUCTION_PROTOCOL.md
- ../models/qpadm_model_specifications.tsv
- ../data/qpadm_target_and_sources.txt
- ../data/qpadm_right_set.txt
- ../outputs/raw/illustrativedna/

The raw output files and screenshots should be used to verify that the submitted values match the web-platform results.
