# Response to PCI Evol Biol Managing Board

Dear PCI Evol Biol Managing Board,

Thank you for the detailed guidance on the submission entitled "Ashkenazi DNA Under the Microscope: Debunking the Europe-Levant Intermediate Myth." I have revised the manuscript and repository to address the reproducibility and methods concerns raised in your message.

First, I have rewritten the Materials and Methods section in a standard population-genetic format. The revised section now identifies the data resources, target populations, source populations, right-set populations, comparator sets, software or web-platform environment, SNP-handling settings, and interpretation criteria used for the qpAdm, FST, Global25, and PCA analyses.

Second, I have reorganized the GitHub repository at https://github.com/StevenParker-collab/ashkenazi-dna-pci-reproducibility so that the analyses can be audited from explicit input files, model specifications, raw outputs, derived tables, figures, and web-platform reproduction notes. The repository now contains the exact IllustrativeDNA Admix Lab Human Origins v62.0 target labels, source labels, right-set labels, FST comparator labels, settings, raw output logs, screenshots, and derived table files.

Third, I have added a Data and Software Availability statement and genotype retrieval notes. These explain that the submitted qpAdm and FST outputs were generated using the IllustrativeDNA Admix Lab Human Origins v62.0 web environment. No local qpAdm or FST scripts were used for the submitted analyses. The repository records the exact target, source, right-set, comparator definitions, and platform settings needed to repeat the runs in the same web interface.

Fourth, I have added a reviewer checklist so that the repository can be inspected systematically. The checklist directs reviewers to the data-source notes, model specification tables, raw output files, Global25 coordinates, derived tables, and web-platform reproduction notes and utility scripts.

I appreciate the opportunity to revise the submission and hope that the revised manuscript and repository now address the points raised by the Managing Board.

Sincerely,

Steven Parker
