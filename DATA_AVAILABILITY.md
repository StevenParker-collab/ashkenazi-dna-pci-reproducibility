# Data and software availability

No newly sequenced genotype data are reported in this manuscript. The qpAdm and FST analyses reported in the manuscript were run through the IllustrativeDNA Admix Lab web interface using the Human Origins v62.0 dataset.

IllustrativeDNA Admix Lab URL:
https://illustrativedna.com/admix-lab/

The reported qpAdm and FST results were generated by selecting population labels and settings in the web interface. No local qpAdm, ADMIXTOOLS, or FST shell scripts were used to generate the submitted qpAdm and FST outputs.

## Human Origins v62.0 analyses

Dataset selected in the web platform: Human Origins v62.0
Platform: IllustrativeDNA Admix Lab
Analyses: qpAdm and FST

The repository records the exact population labels used in the web platform:

- data/qpadm_target_and_sources.txt

- outputs/raw/illustrativedna/illustrativedna_qpadm_north_italian_model_raw_output.txt

The FST platform output reported use of HO.geno with 21,945 samples and 584,131 SNPs, with allele frequencies calculated from 118 samples in 13 populations. The FST run used Adjust Pseudohaploids = NO. The qpAdm runs used allsnps = TRUE.

## Global25 analyses

Modern Global25 analyses used scaled modern Global25 population-average coordinates. The two target coordinates used in the manuscript are provided in:

data/global25_targets.csv

The coordinate source described in the manuscript is the Davidski Global25 modern scaled population-average dataset, made available through the Eurogenes/Global25 distribution referenced by the author. The distance script requires a comparator coordinate file at:

data/external/modern_scaled_population_averages.csv

After the comparator file is placed there, the Global25 distance rankings can be reproduced with:

python scripts/01_reproduce_global25_distances.py --target Ashkenazi_Germany
python scripts/01_reproduce_global25_distances.py --target Italian_Jew

## Reproducibility scope

The submitted qpAdm and FST analyses are reproducible inside the same IllustrativeDNA Admix Lab Human Origins v62.0 web environment using the exact population labels and settings documented here. This repository does not claim that the original qpAdm and FST outputs were produced by local command-line scripts.
