# Notes on PCI example paper format

The example paper provided by PCI uses a standard format that this repository attempts to follow:

- It names the genotype dataset and version.
- It reports sample counts and SNP counts.
- It describes filters and exclusions.
- It names the software and parameters used.
- It provides supplementary files for qpAdm, D-statistics, FST, and related outputs.
- It provides a data availability statement with public genotype access.
- It provides code to reproduce figures.

This repository mirrors that structure by adding METHODS.md, DATA_AVAILABILITY.md, model specification TSV files, raw output logs, derived tables, and scripts. The remaining point to verify is public access to the exact Human Origins v62.0 genotype files used by the web platform or a local rerun using a public genotype release.
